# Base Defender

WIP roguelike arena shooter inspired by Brotato.

You defend a fortified base against endless waves of enemies using modern weapons. Survive as long as possible by crafting powerful weapon builds and reinforcing your defenses between waves.

### Core Loop
- Real-time top-down combat
- Kill enemies to collect resources
- Spend resources in the shop to buy and upgrade weapons + base defenses
- Push for longer survival runs with stronger synergies

### Planned Features
- Multiple modern weapons (assault rifles, shotguns, SMGs, snipers, rocket launchers, etc.)
- Deep weapon stats, attachments, and synergies
- Base building with turrets, barricades, and traps
- Character traits and passive upgrades
- Enemy variety and mutations
- Meta progression system

The game hasn't been worked on yet. This repo is just for planning and early development.
